## 💡 KEY CONCEPTS TO REMEMBER

### 1. Google Sign-In Button
```jsx
<GoogleSignInButton variant="primary" size="md" fullWidth />
```
Just one line to add Google OAuth to any page!

### 2. Access User Data
```jsx
const { userData, isAuthenticated, isAdmin } = useAuth();
```
Use anywhere, always current!

### 3. CRITICAL: Always Include userId in Orders
```javascript
const orderData = {
  userId: user.uid,  // ← NEVER forget this
  items,
  totalAmount
};
```
Without this, orders won't be queryable by user!

### 4. Get User's Orders
```javascript
ordersService.getAll(user.uid)  // Returns only user's orders
```
Simple, clean, and filtered!

### 5. Session Persists Automatically
```jsx
// In App.js
useAuth();  // That's it! Session restored on refresh
```
One line, everything works!
